About Mayura

Sarvam Mayura is a bidirectional translation model supporting English and 10 major Indian languages. It handles style control (formal / colloquial), script switching (Roman / native), numerals formatting, and speaker gender — all in a single API call.

Powered by Sarvam AI · CORS via cors.sh · Add a free cors.sh key above to lift rate limits.
